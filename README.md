# Car-Data-Analysis-using-Python-EDA-Project-
This project focuses on analyzing a car dataset to extract meaningful insights using Python. The goal is to perform data cleaning, transformation, and exploratory data analysis (EDA) to understand patterns like fuel efficiency, weight distribution, origin-wise trends, and more.
🎯 Objectives
Clean and preprocess raw car dataset
Handle missing values
Perform feature transformation
Analyze relationships between variables
Generate insights using visualizations
📊 KPIs (Key Performance Indicators)
🚘 Average MPG (Miles Per Gallon)
⚙️ Average Engine Cylinders
🌍 Distribution of Cars by Origin (Asia, Europe, USA)
⚖️ Average Weight of Cars
📈 MPG improvement trends
🔍 Count of cars based on features
🔧 Features / Tasks Performed
✔️ Data Cleaning
Handled missing values using fillna()
✔️ Data Filtering
Used conditions (isin(), logical operators)
✔️ Feature Engineering
Modified MPG values using apply()
✔️ Data Analysis
Compared weight, origin, and fuel efficiency
✔️ Lambda Functions
Applied transformations efficiently
